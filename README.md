# MadeInUzbekistan.org

Initial scaffold for a B2B export directory MVP.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- Postgres
- Resend

## First delivery focus
- `/find-suppliers-in-uzbekistan`
- inquiry capture
- lead persistence
- email notification
- minimal internal leads page

## Environment
Copy `.env.example` to `.env` and fill values.

## Commands
```bash
npm install
npm run dev
npx prisma generate
```
